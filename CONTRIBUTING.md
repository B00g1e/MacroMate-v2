# Contributing to MacroMate

Thank you for your interest in contributing to MacroMate!

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Development Setup

### Android Client
1. Open `android-client` in Android Studio
2. Sync Gradle
3. Run on device or emulator

### PC Server
1. Navigate to `pc-server`
2. Run `mvn clean package`
3. Test with `java -jar target/macromate-server-1.0-SNAPSHOT.jar`

## Code Style

- Java: Follow standard Java conventions
- JavaScript: Use modern ES6+ syntax
- Indentation: 4 spaces for Java, 2 spaces for JS/HTML/CSS

## Reporting Bugs

Open an issue with:
- Clear description
- Steps to reproduce
- Expected vs actual behavior
- Environment details (OS, Java version, Android version)

## Feature Requests

Open an issue describing:
- The feature
- Use case
- Proposed implementation (optional)

## Questions?

Feel free to open an issue for any questions!
